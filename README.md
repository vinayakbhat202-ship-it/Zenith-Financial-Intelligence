# Zenith — Enterprise Forensic Auditing Command Center

Zenith is a multi-role portfolio project that integrates unsupervised machine learning, statistical modeling, and agentic AI (Retrieval-Augmented Generation) to perform automatic, real-time forensic auditing of corporate transactions.

## Features
*   **Forensic Ledger Tab:** AI/ML-driven detection of transaction anomalies and automatic policy compliance auditing.
*   **Brand Sentiment Tab:** Multilingual sentiment analysis tracking customer feedback on platform performance.
*   **Market Compliance Tab:** Scraped competitor product pricing feeds detecting "drip pricing" (hidden fees added at checkout).

## Technology Stack
*   **Backend:** FastAPI (Python), SQLite
*   **AI/ML:** Isolation Forest (Scikit-Learn), Benford's Law distribution analysis, Cosine Similarity (local TF-IDF vector database), and OpenAI GPT compliance agent.
*   **Frontend:** Vanilla JS, Tailwind/Glassmorphism CSS, Chart.js
